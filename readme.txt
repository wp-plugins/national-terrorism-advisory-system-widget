== Description ==

<p>This widget Places the National Terrorism Advisory System widget from the Department of Homeland Security on your website. 
The widget is automatically updated as the Department of Homeland Security's <a href="http://www.dhs.gov/files/programs/ntas.shtm">NTAS website</a>) issues advisories.</p>

<p>The following aspects are customizeable:
<ul>
<li>Widget title</li>
<li>NTAS Widget width in pixels</li>
<li>NTAS Widget height in pixels</li>
</ul>
</p>
<p>Clicking the widget pops open a new window to the NTAS advisory page: <a href="http://www.dhs.gov/files/programs/ntas.shtm#current">http://www.dhs.gov/files/programs/ntas.shtm#current</a>.</p>

== Installation ==

1. Upload `national-terrorism-advisory-system-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to one of your theme's widget areas.
